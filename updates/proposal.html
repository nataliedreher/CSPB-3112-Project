Text-Based Adventure Game

Vision statement

I will be making a simple Text-Based Adventure Game Android application which will ideally be made available in Google Play Store.

Motivation

The first goal of this project is to learn the basics of Java. Java is an incredibly popular language that many companies use so it makes sense to gain experience with this language. The second goal is to learn the basics of Android app development. Many companies utilize android applications so having this experience would also be beneficial for me.

Specific and measurable goals

The first 6-8 weeks I will dedicate solely to learning Java. Freecodecamp.com and Coursera have free Java learning courses available which I will be utilizing. After the first 6-8 weeks I will add in courses related to android development which will also build on the Java courses. Android.com has free learning resources which should prove helpful.

Risks to project completion

There is one main risk to this project itself, which is that I’m committing to learning two separate things, Java and Android studio. It might be that I’m taking on too much for this project. The other main risk is related to time management. I am enrolled in two other classes, both of which also have project which means I will have to balance finishing three separate projects at once.

Mitigation Strategy

  1. Be proactive. I need to make sure to hit all my goals quickly and not fall behind on the language learning.
  2. Keep it simple. Since the goal of this project is learning Java and Android development, I need to make sure aspects of this project that are unrelated to these goals are kept as simple as possible. To that end I plan on using either Novel.ai or ChatGPT to write the story aspects. I also plan on having as basic of a UI as possible.
  3. Utilize multiple learning resources. Online Java resources are widely available and many seem to be quite thorough. If I’m not getting enough knowledge related to this project out of the above resources, I’ll need to seek out additional resources.

Project Assessments

  • The largest measure of success for my project will be to have a completed app that is available in the android app store.
  • Another important measure of success will be ensuring the app works fairly well and is at least somewhat presentable.
  • The last measure of success is for the app to be at least somewhat fun to play. No one wants to play a terrible game.
